05 — Princípios de design

Dignidade antes de dado
Cada número representa uma pessoa. Linguagem acessível, sem jargão clínico.
Cor com propósito
Verde = dentro do prazo. Laranja = atenção. Vermelho = crítico. Consistente em todo o sistema.
Número grande, contexto pequeno
A métrica central é sempre a maior coisa na tela. O texto explica o significado.
Editorial, não técnico
Serif para impacto emocional. Sans-serif para informação. Espaço vazio como respiração.
